Overview

Civic DataDog is a static public transparency portal for Minnesota state government spending. This page provides a readable, link-rich overview of Minnesota state expenditures using official public data from the State of Minnesota and the federal government.

All numbers on this site should be treated as summaries of official records. For legal, audited, or most up-to-date figures, always refer to the primary sources listed on the Sources page.

Planned Static Tables

The following sections describe static tables that should be generated offline from CSV exports and then embedded here as pure HTML. Each table should point back to the exact CSV or report it was derived from.

  • State Payments (SWIFT) – Top vendors, payments by category, payments by agency.
  • State Contracts & Grants – Contract totals by vendor, open vs. closed contracts, grant distributions.
  • State Payroll – Aggregated totals by agency and job classification (no individual names unless already public in source data).
  • Local Government Spending – County, city, and special district totals from State Auditor reports.

All tables should be generated from official CSVs or PDF-to-CSV conversions, then reviewed before publication.

Key Minnesota Data Links

  • TransparencyMN (SWIFT) – State payments, contracts, grants, and payroll.
  • Minnesota Management & Budget (MMB) – Biennial budgets and historical expenditure tables.
  • Office of the State Auditor (OSA) – County and local government audited spending.
  • USAspending.gov – Federal funds flowing into Minnesota (filter by state = MN).

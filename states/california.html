California Government Spending Data Sources

Access official California government spending data through the following authoritative sources. These portals provide state budget information, spending records, transparency data, and public records for California.

All data sources are official government websites. For federal funds flowing into California, see USAspending.gov filtered by state.

Official California Data Sources

  • State Transparency Portal / Open Data - California state transparency portal and open data access
  • State Auditor / Comptroller - California state auditor office with audited spending reports and financial records
  • Budget Office / Management & Budget - California state budget office with budget documents and spending reports
  • USAspending.gov - Federal funds flowing into California (grants, contracts, assistance) - Filter by state: CA

About California Spending Data

California government spending data includes state budget information, agency spending records, contracts, grants, and local government financial data. All data is compiled from official California state sources and federal spending databases.

For comprehensive information about government spending transparency, see the Sources page. For methodology on how data is collected and published, see Methodology.
